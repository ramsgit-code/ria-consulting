# Prompts Kling — Antes / Después (Go High Level)

## Ajustes en Kling (igual para los 6 videos)

| Campo | Valor |
|-------|--------|
| Aspect ratio | **16:9** |
| Duración | **10 s** (si sale caro, 5 s) |
| Modo | **Professional** / alta calidad |
| Cámara | **Slow push-in** o **static** (sin movimientos bruscos) |
| Consistencia | Genera ANTES y DESPUÉS el mismo día; misma relación de aspecto |

### Negative prompt (pegar en todos)

```
blurry text, unreadable UI, fake logos, watermark, cartoon, anime, 3d render look, distorted hands, extra fingers, shaky camera, fast cuts, comedy, stock photo smile, cluttered desk, neon cyberpunk, low resolution, oversaturated colors, deformed laptop screen
```

## Tips post-generación

1. Si el texto de la UI sale ilegible: acércate menos a la pantalla en un segundo clip o difumina ligeramente el monitor en CapCut y superpone captura real de GHL.
2. Para coherencia antes/después: sube un frame del ANTES como **image reference** al generar el DESPUÉS (misma persona, misma sala).
3. Música: Epidemic Sound / Artlist — corporate uplifting, sin letra.
